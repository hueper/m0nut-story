Story Map Buttons

E.S. (Eine Sache)

Audiovisuell:
Bezug zu 68 / Elefanten / Elfenbein / Aramith / Murmelbahn?

Performativ:
Den Kosmonauten wird durch die Performer von außen ein Objekt in das Cockpit materialisiert: Bei dem Objekt handelt es sich um eine ((hohle) Billard-)Kugel. Die Materialisierung geschieht durch "Hineinrollen". Die Kugel wird an einem Startpunkt (außen / unterhalb der Bühne) auf eine Bahn gelegt, die sie dann entlangrollt, bis sie vor / zwischen den Kosmonauten zum Stehen kommt.
Die Materialisierung des E.S. wird durch Drücken des E.S.-Buttons eingeleitet. (Bis zur Dematerialisierung leuchtet die Kugel immer heller. Steht die DM unmittelbar bevor, leuchtet E.S. sofort ganz hell.)
